About morris traversal

It aims at traversing an binary tree in inorder without using stack.

The main idea: Create links to the successor(use leaf node's left or right null pointer) Print the data using these links Revert the changes to restore original tree.

Inorder Example

Let's explain inorder traversal as an example: If the current node has left child, find the predecessor of current node, set current as its right child. In this way, we can return to parent after finish traverse the left subtree.

The presudo code is below:

{% highlight lisp linenos%} Initialize current as root While current is not NULL If current does not have left child a. Print currentâ€™s data b. Go to the right, i.e., current = current->right Else a. Find the predecessor of current node b. If predecessor's right child is null, 1) make current the right child of its predecessor 2)Go to this left child, i.e., current = current->left Else: 1) if predecessor's right child is current node, set predecessor's right child to null 2) print current node, set current node to its right child {% endhighlight %}

Jave implementation:

{% highlight java linenos%} public void morrisTraversal(TreeNode root){ if(root == null) return; TreeNode pre = null; while(root != null){ if(root.left == null){ System.out.println(root.val); root = root.right; }else{ /* find predecessor */ pre = root.left; while(pre.right != null && pre.right != root) pre = pre.right; /* if predecessor's right == null, set current node as its right child */ if(pre.right == null){ pre.right = root; root = root.left; }else{ /* recover tree, when return to parent node the second time */ pre.right = null; System.out.println(root.val); root = root.right; } } } } {% endhighlight %}

Analysis:

To find predecessor, we visit a node at most 3 times, so the overall time complexity is O(n)

Preorder and Postorder

The morris traverse can be applied to preorder traverse and postorder traverse with some changes.

preorder

It's very similar with inorder, just change the print out sequence.

The presudo code is below:

{% highlight lisp linenos%} Initialize current as root While current is not NULL If current does not have left child a. Print current node b. Go to the right, i.e., current = current->right Else a. Find the predecessor of current node b. If predecessor's right child is null, 1) make current the right child of its predecessor 2) Print current node 3)Go to this left child, i.e., current = current->left Else: 1) if predecessor's right child is current node, set predecessor's right child to null 2) set current node to its right child {% endhighlight %}

java implementation:

{% highlight java linenos%} public void morrisTraversal(TreeNode root){ if(root == null) return; TreeNode pre = null; while(root != null){ if(root.left == null){ System.out.println(root.val); root = root.right; }else{ /* find predecessor */ pre = root.left; while(pre.right != null && pre.right != root) pre = pre.right; /* if predecessor's right == null, set current node as its right child */ if(pre.right == null){ pre.right = root; System.out.println(root.val); root = root.left; }else{ /* recover tree, when return to parent node the second time */ pre.right = null; root = root.right; } } } } {% endhighlight %}

We need to create a dummy node and set root as its left child.

The presudo code is below:

Recover binary tree use morris traverse

Two elements of a binary search tree (BST) are swapped by mistake. Recover the tree without changing its structure.

We can easily solve this problem by recursion or using stack. What should we do if it requires us to use constant space? Mirros Traverse!

java implementation:

{% highlight java linenos%} package leetcode; /** * @author wish * */ public class BinaryTree_RecoverBinarySearchTree { /* * O(1) space */ public void recoverTree(TreeNode root) { /* arr[0] arr[1] stroe two swapped elements, arr[2] store pre.*/ TreeNode[] arr = new TreeNode[2]; morrisTraversal(root, arr); int temp = arr[0].val; arr[0].val = arr[1].val; arr[1].val = temp; } public void morrisTraversal(TreeNode root, TreeNode[] arr){ if(root == null) return; TreeNode pre = null; /* predecessor*/ TreeNode previous = null; while(root != null){ if(root.left == null){ if( previous != null && previous.val > root.val){ if(arr[0] == null) arr[0] = previous; arr[1] = root; } previous = root; root = root.right; }else{ /* find predecessor */ pre = root.left; while(pre.right != null && pre.right != root) pre = pre.right; /* if predecessor's right == null, set current node as its right child */ if(pre.right == null){ pre.right = root; root = root.left; }else{ /* recover tree, when return to parent node the second time */ pre.right = null; if( previous != null && previous.val > root.val){ if(arr[0] == null) arr[0] = previous; arr[1] = root; } previous = root; root = root.right; } } } } } {% endhighlight %}
